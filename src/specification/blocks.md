# blocks
blocks start with `blk :blockname` and end with `end :blockname`.
| symbol    | description                                                     |
| --------- | --------------------------------------------------------------- |
| `:arg`    | arguments                                                       |
| `:var`    | all variables that must be taken out of context of the function |
| `:{name}` | named block, can be jumped to, otherwise is skipped             |

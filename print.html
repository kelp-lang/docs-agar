variables

reference a variable with %varname.

global variables

there are some global variables | name | description | | -------- | ------------- | | %true | boolean true | | %false | boolean false |

unnamed variables

variables don't need to have names, you can create an unnamed variable by providing number instead of varname.

set %0 0
add %0 1
// now %0 is 1

blocks

blocks start with blk :blockname and end with end :blockname. | symbol | description | | --------- | --------------------------------------------------------------- | | :arg | arguments | | :var | all variables that must be taken out of context of the function | | :{name} | named block, can be jumped to, otherwise is skipped |
